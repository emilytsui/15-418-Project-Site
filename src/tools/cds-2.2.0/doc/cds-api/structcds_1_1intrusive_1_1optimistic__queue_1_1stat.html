Detailed Description

template<typename Counter = cds::atomicity::event_counter>
struct cds::intrusive::optimistic_queue::stat< Counter >

OptimisticQueue internal statistics. May be used for debugging or profiling

Template argument Counter defines type of counter. Default is cds::atomicity::event_counter. You may use stronger type of counter like as cds::atomicity::item_counter, or even integral type, for example, int.
