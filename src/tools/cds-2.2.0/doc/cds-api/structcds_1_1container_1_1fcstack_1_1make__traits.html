template<typename... Options>
struct cds::container::fcstack::make_traits< Options >

Metafunction converting option list to traits.

Options are:

  • any cds::algo::flat_combining::make_traits options
  • opt::stat - internal statistics, possible type: fcstack::stat, fcstack::empty_stat (the default)
  • opt::enable_elimination - enable/disable operation elimination By default, the elimination is disabled.
